
# Number Guessing Game

A fun Python game where the player tries to guess a random number between 1 and 20.  
This project was created to practice loops, conditionals, and random number generation.

---

# Features
- Random number between 1 and 20  
- Unlimited guesses  
- Feedback: too high / too low  
- Tracks number of attempts  

---

# What I Learned
- Using the `random` module  
- While loops  
- Conditionals  
- Game logic  
- Counting attempts  

---

# How to Play

Run the game using:

```bash
python number_guessing_game.py
